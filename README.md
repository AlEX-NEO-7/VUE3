## diff核心
1. 先比较旧元素是否能复用 div -> span （不能复用删除div创建span）
2. 如果两个元素类型都是div key都是undefined， 说明两个元素可以复用。需要对比两个元素的属性
3. 比较新旧两个节点的子元素,分六种情况讨论。
